# noco-example-output

This is an example output from the (NoCo platform)[https://noco.io]

the api is under the subfolder `api`

# To start the api 
```
cd api
# Please make sure you have a sql instance to connect to! It's set up for mysql out of the box. 
npm i
npm run dev
```

# To start the app 
```
cd app
npm i
npm run dev
```
